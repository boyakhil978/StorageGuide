# StorageGuide Roadmap

- Add an operator menu for easier GUI management of server settings.

- Add operator settings to:
  - Enable or disable the sloppiness detector.
  - Require clients to install StorageGuide.
  - Configure the sloppiness announcement cooldown.

- Add a cell editor option to exclude a chest from sloppiness detection.

- Add a public sloppiness history menu:
  - Show all detected instances, including events hidden by announcement cooldowns.
  - Group or sort instances by player.
